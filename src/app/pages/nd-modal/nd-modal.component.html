Modal Implementation Guide

Step 1: Open Modal

  1. Use Angular Material MatDialog service to open the modal.
  2. Pass data as header, data, and type.
  3. The modal supports 'medium' and 'dynamic' views.

Step 2: Required Inputs

  • ndModalType: 'medium' or 'dynamic'
  • getHeaderValue: title to show in header
  • getDataValue: array of (title, description )

Step 3: Render Modal

Once inputs are received, the HTML sections are displayed conditionally using *ngIf. Actions available:

  • Submit → triggers onSubmit()
  • Cancel → triggers close()

Step 4: Close Modal

  1. Use the top right close icon or Cancel button.
  2. Ensure the close() method triggers dialogRef.close() to terminate the popup safely.
