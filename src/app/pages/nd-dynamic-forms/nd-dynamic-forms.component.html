Dynamic Reactive Form Builder — Implementation Guide

Overview

The Dynamic Reactive Form Builder is a reusable Angular 16 component that generates fully reactive forms at runtime using a JSON configuration (DynamicField[]). It supports both Bootstrap and Angular Material UI modes.

Supported UI Modes

  • Bootstrap Mode – Uses native inputs, selects, and layout grid.
  • Material Mode – Uses Angular Material components (mat-form-field, mat-select, mat-radio, etc).

Switch modes using: @Input() type: 'bs' | 'material'

Key Features

  • Supports text, number, email, password, textarea, date, select, multiselect, radio, checkbox, and file inputs.
  • JSON-driven field rendering using *ngFor and ngSwitch-style logic.
  • Reactive validation using Angular Validators.
  • Conditional visibility, disable, and readonly logic.
  • Role-based permissions support.
  • Bootstrap + Angular Material compatible.
  • Emits structured data on submit.

Validation & Conditions

  • Uses Angular Validators (required, minLength, maxLength, email, pattern, etc).
  • showWhen → Controls visibility dynamically.
  • disableWhen → Enables/disables fields dynamically.
  • readonlyWhen → Toggles readonly state.

Submit Behavior

  • Form is marked touched if invalid.
  • Valid submission emits raw form value:
    • this.Submitted.emit(this.form.getRawValue())

Extending the Builder

  1. Add new type in DynamicField.
  2. Update the template for Bootstrap & Material blocks.
  3. Register any special validation in createForm().

Typical Use Cases

  • Admin configuration panels
  • Role-based forms
  • Survey / Questionnaire builders
  • Dynamic filter forms
  • Metadata-driven UI forms
