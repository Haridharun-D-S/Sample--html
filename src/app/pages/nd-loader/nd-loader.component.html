Loader Setup Guide

Step 1: Clone the Repository

  1. Access the repository from: https://gitlab.com/ecare-project/framework-ui
  2. Ensure you have access permissions.
  3. Clone using Node.js version 20.
  4. Branch: dev-ui

Step 2: Component Integration

  1. Import LoaderComponent in your module or shared module.
  2. Use the loader in any component HTML: 
    <app-loader [message]="'Loading data...'"></app-loader>
  3. The loader will display a small box with text. You can customize text through the [message] input.

Note: Add Loader GIF

    Place your loader image file inside the assets folder at: src/assets/img/loader-svam.gif
