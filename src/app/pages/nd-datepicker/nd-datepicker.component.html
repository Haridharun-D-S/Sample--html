Datepicker Setup Guide

Introduction

The Datepicker Component provides flexible date selection options, including single-date selection, range selection, filtering (weekends, etc.), and datepicker with custom action buttons.

Available Variants

  1. Basic Datepicker: Allows single date selection with min/max limits.
  2. Date Range Picker: Allows users to select a range (start and end dates).
  3. Datepicker with Filter: Disables weekends or specific days.
  4. Datepicker with Action Buttons: Includes Apply and Cancel actions.
  5. Date Range (From - To): Two separate fields for from/to date selection.
