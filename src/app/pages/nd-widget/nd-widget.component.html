Widget Setup Guide

Step 1: Clone the Repository

  1. Access the repository from the following URL: https://gitlab.com/ecare-project/framework-ui
  2. Ensure you have the necessary permissions to access the repository.
  3. Clone the repository to your local environment using Node.js version 20.
  4. Branch: dev-ui

Step 2: Install Dependencies

Install all required dependencies listed below. If dependency conflicts occur, use either --legacy-peer-deps or --force during installation.

  • @novacisdigital/nd-custom-grid: ^1.5.7 — to run Demo
  • @novacisdigital/nd-custom-highcharts: ^1.3.1 — to run Demo
  • bootstrap: ^5.3.8
  • crypto-js: ^4.1.1 — to run Demo

Optional: If you still encounter dependency issues, remove package-lock.json and the node_modules directory, then reinstall using npm install.

Step 3: Component Integration

  1. Run the application to ensure all dependencies and modules are loaded correctly.
  2. Navigate to the widget component — this section contains all relevant code, configuration, and logic associated with the widget functionality.
  3. Optional: Visit http://localhost:4200/pages/demo to view widgets displayed with sample (dummy) data.
