Splitter Component Overview

The Reusable Splitter Component provides a horizontally or vertically resizable layout for Angular applications. It allows complete customization via input bindings. The splitter supports dynamic panel sizes, minimum size constraints, gutter sizing, and lock movement all controlled by user-defined inputs.

This component is built using Angular and angular-split, without external UI frameworks.

HTML Template

This structure defines the split layout with two panels, left and right, whose sizes are controlled by inputs. Minimum sizes ensure panels do not collapse during drag events.

Key Layout Features:

  • Wrapper for the full splitter container.
  • Resizable left and right panels.
  • Configurable initial panel sizes in percentage.
  • Minimum panel sizes enforced in pixels.
  • Gutter (drag handle) size adjustable.
  • Ability to lock the splitter movement.
  • Drag events emit start and end callbacks.

Inputs

Input Type Default Description
direction 'horizontal' | 'vertical' horizontal Split direction
leftSize number 30 Initial left panel size (%)
rightSize number 70 Initial right panel size (%)
gutterSize number 6 Size of the drag handle in pixels
restrictMove boolean false Lock splitter movement
minLeftPx number 50 Minimum left panel size in pixels
minRightPx number 50 Minimum right panel size in pixels

Outputs

Output Type Description
dragStart EventEmitter<void> Emits when dragging starts
dragEnd EventEmitter<void> Emits when dragging ends (panels clamped to min sizes)

Usage Example

<app-splitter
  [direction]="'horizontal'"
  [leftSize]="30"
  [rightSize]="70"
  [gutterSize]="8"
  [minLeftPx]="150"
  [minRightPx]="150">

  <div left>
    LEFT PANEL CONTENT
  </div>

  <div right>
    RIGHT PANEL CONTENT
  </div>

</app-splitter>

Notes

  • Minimum panel sizes prevent the splitter from collapsing panels to zero width/height.
  • Works for both horizontal and vertical orientations.
  • restrictMove locks the splitter if needed.
  • Panels sizes are in percentage, while min sizes are enforced in pixels.
  • Can be used in dashboards, editors, or any resizable layout.
