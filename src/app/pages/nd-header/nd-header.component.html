Header Component Overview

The Reusable Header Component provides a flexible and dynamic header layout for Angular applications. It allows complete customization via JSON configuration or input bindings. The header supports dynamic menus, search functionality, user information display, and logout actions all controlled by boolean flags and user-defined inputs.

This component is built using pure HTML, CSS, and Angular, without external UI frameworks like Bootstrap or Angular Material.

HTML Template

This structure defines the header layout with optional elements that are shown or hidden dynamically.

Key Layout Features:

  • Wrapper for the full header bar.
  • Holds the logo image and title text.
  • Conditionally shown search input with a search icon.
  • Dynamic menu icons generated from JSON or input array.
  • Displays welcome text and logout icon based on user preferences.

CSS Styles

The styling ensures a clean, modern, and responsive header design with hover effects and flexible layouts.

Key Style Features:

  • Flex layout for evenly spaced alignment.
  • Soft hover effects for interactivity.
  • Icons aligned with text and inputs.
  • Clear separation between menus, search, and user section.

This logic controls the visibility, menu interaction, and event emission of the header component.

Example Usage

Integrate this component into your parent template as shown below:

<app-header-apps
  [title]="'Sapphire Vantage'"
  [imgLogoPath]="'assets/img/logo.png'"
  [fullName]="'User'"
  [menus]="menuItems"
  [isSearchEnabled]="true"
  [isMenuEnabled]="true"
  [isUserInfoEnabled]="true"
  [isLogoutEnabled]="true"
  (menuClick)="handleMenuClick($event)"
  (logout)="onLogout()"
  (searchChange)="onSearch($event)">
</app-header-apps>
